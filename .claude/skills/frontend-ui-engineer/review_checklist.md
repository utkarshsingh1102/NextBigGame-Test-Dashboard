# UI Review Checklist

Before finalizing UI check:

## Layout

Is the content centered?

Are cards aligned in grid?

Is spacing consistent?

## Visual Hierarchy

Is the page title clearly visible?

Are section titles readable?

Are cards visually grouped?

## UX

Is the primary action obvious?

Is navigation clear?

Are hover states present?

## Accessibility

Are buttons keyboard accessible?

Are icons labeled?

Is contrast readable?

## Performance

Are images lazy loaded?

Are components reusable?

Avoid duplicate code.
