# UX Rules

Every UI must pass these UX principles.

## Visual Hierarchy

Use 3 levels of hierarchy.

Level 1
Page title

Level 2
Section titles

Level 3
Card information

Avoid using same font weight everywhere.

## Navigation UX

Sidebar must contain:

Primary navigation
Secondary navigation
User utilities

Selected page should have:

highlight background
left border indicator

## Card UX

Cards should contain:

image or icon
title
metadata
action buttons

Padding

p-4 or p-6

Avoid cramped content.

## Button Hierarchy

Primary Button
bg-black text-white

Secondary Button
border border-gray-300

Tertiary Button
text-gray-500

Primary action must be visually strongest.

## Empty States

Always design empty states.

Example:

No games found  
Try adjusting filters.

## Loading States

Add skeleton loaders for

cards
tables
lists
